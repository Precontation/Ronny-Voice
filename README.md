# Ronny Voice

## About

Ronny Voice is a open-source, private, and AI-powered voice assistant for the 2026 Invention Convention.

## Features

- Fast - utilizes models from Groq, streaming directly into Google's Text-To-Speech, so you hear messages before the message is even done!
- Targets low-end hardware - The target machine for this device is the Raspberry Pi Zero 2 W, meaning it can run on very low-end hardware.

## Privacy

1. There is no telemetry in our product.
2. As stated in Groq's [policies](https://console.groq.com/docs/legal), they have zero retention of data, nor do they train on said data.
3. As stated in Google Cloud's [privacy policy](https://cloud.google.com/privacy), the customer data being shared with Google is opt-in by default. Additionally, exact user messages is never sent to Google Cloud, only the AI responses.
